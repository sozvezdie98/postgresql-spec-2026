# PostgreSQL для разработчиков — Специалист при МГТУ им. Баумана (2026)

Лабораторные работы и проекты по курсу **«PostgreSQL для разработчиков»**  
Учебный центр «Специалист» при МГТУ им. Н.Э. Баумана  


## Темы курса
- Проектирование реляционных БД
- Нормальные формы, индексы, партиционирование
- Сложные SQL-запросы (CTE, оконные функции, JSONB)
- psycopg2 + asyncpg + SQLAlchemy 2.0 + Alembic
- Оптимизация запросов (EXPLAIN ANALYZE)
- Транзакции, блокировки, VACUUM
- Резервное копирование и репликация
